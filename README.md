# octopus-case-study

## Repository Structure

This repository contains:

- **data/** - Directory for CSV documents and data files
- **src/** - Directory for source code
- **schema.sql** - SQL file defining the database structure based on data files

## Getting Started

1. Place your CSV data files in the `data/` folder
2. Add your source code to the `src/` folder
3. Update `schema.sql` to match the structure of your CSV files
4. Import the schema into your database system
